# React Redux Image Search App

A web application built using React and Redux that allows users to search images, save selected images to a personal collection, view them on a collection page, and delete images from the collection. This project demonstrates Redux state management, API integration, and dynamic UI updates.

## Features
- Search images using an API
- Save images to a personal collection
- View saved images on the collection page
- Delete images from the collection
- State management using Redux
- Responsive user interface
